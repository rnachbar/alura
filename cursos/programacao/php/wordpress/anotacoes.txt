############### 1. Configurando o WordPress na Hospedagem ###############

Para acompanhar o conteúdo das aulas, faça o download dos arquivos dos cursos de Wordpress I e II:
https://s3.amazonaws.com/caelum-online-public/wordpress/wordpres-i-e-ii-materiais-alura.zip

Nesta aula vimos a contratação de uma hospedagem gratuita e de uma hospedagem paga. Que características a hospedagem deve possuir para que possamos instalar o WordPress?

Para que o WordPress (versão 4.4.2 – disponível para download no primeiro exercício) funcione corretamente em sua hospedagem, é necessário que ela tenha suporte a PHP na versão 5.6 ou superior e MySQL na versão 5.6 ou superior (ou MariaDB na versão 10.0 ou superior – utilizamos MySQL neste curso).

Há um arquivo no WordPress que devemos configurar para que haja a conexão entre a plataforma e o banco de dados. Que arquivo é esse?

wp-config.php

Quando baixamos o WordPress devemos renomear o arquivo wp-config-sample.php para wp-config.php. Dentro dele devemos inserir as informações necessárias: nome do banco de dados, usuário do banco de dados, senha do usuário e endereço do banco de dados, caso sua hospedagem forneça um diferente do local (localhost).

Após a alocação de todos os arquivos do WordPress, mais a edição adequada do arquivo de configuração, qual é o endereço que devemos acessar pelo navegador para que o WordPress se instale em nosso banco de dados?

"seu domínio('+/pasta/´, caso instalado em outro local)"/wp-admin/install.php

Depois que o wp-config.php está configurado corretamente e todos os arquivos do WordPress estão no servidor, basta acessar "seu domínio"/wp-admin/install.php que uma página será aberta para que você configure o nome do site, seu usuário e sua senha – então o WordPress já estará instalado.

############### 2. Definindo um tema ###############

Nesta aula pesquisamos temas, mas instalamos o FlatBootstrap 1.8.1 por meio de upload para que acompanhemos o mesmo conteúdo dos vídeos (por favor, não atualize o tema durante o curso para que não haja diferença no conteúdo do aprendizado). Comente como um tema pode ser instalado pelo próprio painel do WordPress.

Para instalar um tema a partir do próprio WordPress podemos pesquisar por meio do menu Aparência > Temas. Na parte superior háverá um botão "Adicionar Novo", ao clicarmos nele acessamos uma página com temas disponíveis, onde podemos pesquisar pelo tema que mais se adeque às nossas necessidades.

Nesta aula também instalamos o plugin Jetpack, recomendado pelo tema escolhido. Das alternativas a seguir, qual NÃO representa uma função do Jetpack?

Personalização do CSS do tema

O Jetpack é um plugin dos desenvolvedores do próprio WordPress que permite diversas funções, mas personalização do arquivo de estilos não é com ele, infelizmente. Isso pode ser editado no próprio código ou com algum menu de CSS personalizado do próprio tema.

Iniciamos a configuração da página inicial e, para adicionar elementos, utilizamos o plugin Bootstrap Shortcodes, recomendado pelo tema. O que esse plugin possibilita que façamos?

O plugin Bootstrap Shortcodes utiliza o framework Bootstrap e com ele conseguimos adicionar diversos elementos às nossas páginas, utilizando o próprio editor do WordPress. Ele gera códigos em colchetes, muito parecidos com tags HTML.
